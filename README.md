## アプリケーション概要
このアプリでは、ユーザーが自由に記事を投稿することができます。投稿するにはユーザー登録が必要です。自分が投稿した記事は更新や削除をすることもできます。  

## 制作背景
「今何をしているか」などを投稿する機能を有するアプリケーションに、メモや備忘録としても使用できる機能が備わっていたら便利だと思ったので、このアプリケーションを制作することにしました。
